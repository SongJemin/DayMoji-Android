> 하루하루 자신의 감정을 분석하고 기록하는 앱(Day + Emoji) - 혼자 틈틈이 진행하는 플젝



---

## 1. 커밋 컨벤션

#### 1) 타입(type) : 제목(subject) - 필수

1. feat : 새로운 기능 추가

2. fix : 버그 수정

3. build : 빌드 관련 파일 수정(ex : builde.gradle, manifest)

4. ci : CI 관련 설정 수정

5. style : ui 수정

6. docs : 문서 수정

7. refactor : 코드 리팩토링

8. test : 테스트 코드 수정

9. release: 릴리즈에 대한 커밋

10. chore : 그외 자잘한 수정



#### 2) 내용(body) - 선택

1. 간략 설명
    - 상세 설명
